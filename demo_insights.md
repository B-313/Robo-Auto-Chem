# Demo Insights — Traffic Light Colour Change Experiment

## Automated Colour Change Reaction Workflow

```
┌─────────────────────────────────────────────────────┐
│  1. PREPARATION                                     │
│     - Load vials (pre-filled with reagent)          │
│     - Confirm robot home position                   │
│     - Confirm camera + stirrer connected            │
│     - Run ROI calibration (once per session)        │
└───────────────────┬─────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────┐
│  2. FOR EACH VIAL (i = 0 → 3)                       │
│                                                     │
│  [PICK]                                             │
│     Robot picks vial from rack                      │
│                                                     │
│  [TRANSFER]                                         │
│     Robot moves vial to stirring plate              │
│     Gripper releases                                │
│                                                     │
│  [REACT + OBSERVE]                                  │
│     Stirrer ON at set RPM                           │
│     Camera records (ROI locked to vial)             │
│     Colour state logged: RED → YELLOW → GREEN       │
│     Stirrer OFF after STIR_SECONDS                  │
│                                                     │
│  [RETURN]                                           │
│     Robot picks vial from stirrer                   │
│     Returns to rack slot i                          │
│     Gripper releases                                │
└───────────────────┬─────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────┐
│  3. DATA OUTPUT (per vial)                          │
│     - MP4 video with ROI box overlay                │
│     - CSV: timestamp, colour state transitions,     │
│            elapsed time, pixel counts               │
└───────────────────┬─────────────────────────────────┘
                    │
                    ▼
┌─────────────────────────────────────────────────────┐
│  4. POST-RUN ANALYSIS                               │
│     - Compare colour change timing across vials     │
│     - Flag anomalies (no change, wrong sequence)    │
└─────────────────────────────────────────────────────┘
```

---

## Key Chemistry Workflow Principles

| Principle | Implementation |
|---|---|
| Traceability | Every vial has a timestamped video + CSV log |
| Reproducibility | Fixed RPM, fixed stir time, fixed ROI per session |
| Safety interlock | Stirrer always OFF before robot moves vial |
| Fault tolerance | One vial failure does not stop the run |

---

## Stirring Plate Parameters

| Parameter | Value | Set in |
|---|---|---|
| Stirring speed | `STIR_RPM = 600` RPM | `2026_04_14_routine_cam_no stirring.py` |
| Stir duration | `STIR_SECONDS = 180` s | `2026_04_14_routine_cam_no stirring.py` |
| Temperature | Not yet set (heat control available in `stirring_plate.py`) | `IKADriver.setHeat(temp)` |
| Serial port | `STIR_PORT = /dev/ttyACM0` | `2026_04_14_routine_cam_no stirring.py` |

> To add temperature control, call `plate.setHeat(temp)` and `plate.startHeat()` after `plate.startStir()` in the routine, and `plate.stopHeat()` in the `finally` block alongside `plate.stopStir()`.

## What the CSV Data Maps To

| Colour Transition | Chemical Meaning |
|---|---|
| RED → YELLOW | Induction period |
| YELLOW → GREEN | Reaction completion |

Comparing these times across 4 vials gives you your experimental variable results.
